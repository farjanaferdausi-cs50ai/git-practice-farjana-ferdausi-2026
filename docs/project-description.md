# Project Description: Git-Practice Calculator

 Overview
A simple command-line calculator built in Python for practicing Git version control.
The calculator performs addition, subtraction, multiplication, and division with robust error handling.

Tech Stack
- Python 3.8+
- Git & GitHub

 Features
- Basic arithmetic operations (+, -, *, /)
- Error handling for:
  - Invalid number inputs (non-numeric)
  - Division by zero
  - Unsupported operators
- Clean and simple command-line interface
- Easily extendable utility functions in utils.py

 Project Structure
- src/main.py — Entry point with interactive CLI
- src/utils.py — Pure arithmetic functions
- docs/project-description.md — You're reading it!
- README.md — Repository overview and instructions

 Learning Outcomes
- Practiced Git branching and merging workflows
- Applied clean commit message conventions
- Implemented basic error handling in Python
- Maintained professional file organisation
